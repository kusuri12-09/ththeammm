## 1. 데이터 분석 목적에 맞는 데이터 수집

이 프로젝트의 목적은 Steam 게임의 가격, 무료 여부, 장르, 사용자 평가, 플레이타임, 소유자 수를 활용하여 게임 가격과 인기도 및 만족도 사이의 관계를 확인하는 것이다. 이를 위해 하나의 API만 사용하지 않고, 서로 다른 성격의 두 데이터셋을 수집하였다.

수집된 파일은 다음과 같다.

| 데이터 | 저장 파일 | 주요 컬럼 |
| --- | --- | --- |
| Steam Store API | `data/steam_store_games_001.csv` ~ `data/steam_store_games_005.csv` | `appid`, `name`, `is_free`, `price`, `genres`, `release_date` |
| SteamSpy API | `data/steamspy_games.csv` | `appid`, `name`, `owners`, `owners_avg`, `positive`, `negative`, `average_forever` |

최종 분석에서는 두 데이터를 공통 식별자인 `appid` 기준으로 병합하여 사용하였다. 

- Steam Store 데이터 970행
- SteamSpy 데이터 1000행
- 병합 및 정제 후 최종 분석 데이터 875행

## 2. 결측값과 이상값 확인

데이터 분석 전에 원천 데이터와 병합 데이터에서 결측값과 이상값을 확인하였다. Steam Store 데이터에서는 가격 정보가 없는 행과 장르 정보가 없는 행이 존재하였다. SteamSpy 데이터에서는 주요 분석 컬럼인 소유자 수, 긍정 리뷰 수, 부정 리뷰 수, 평균 플레이타임에 결측값이 없었다.

확인한 주요 결측값은 다음과 같다.

| 데이터 | 컬럼 | 결측 건수 |
| --- | --- | ---: |
| Steam Store | `price` | 93 |
| Steam Store | `genres` | 2 |
| Steam Store | `release_date` | 11 |
| SteamSpy | 주요 분석 컬럼 | 0 |

이상값은 분석 목적에 맞게 다음 기준으로 확인하였다.

- 가격이 음수인 데이터가 있는지 확인하였다. 확인 결과 음수 가격은 없었다.
- 긍정 리뷰 수와 부정 리뷰 수의 합이 0 이하인 데이터가 있는지 확인하였다. 리뷰가 없는 게임은 긍정률 계산이 불가능하므로 제거 대상으로 보았다.
- 무료 게임은 가격이 0일 수 있으므로 가격 0 자체는 이상값으로 처리하지 않았다.
- 가성비 지표는 가격으로 나누는 계산이 포함되므로 무료 게임의 `value_score`는 계산 대상에서 제외하였다.

## 3. 결측값과 이상값 처리

결측값과 이상값은 분석 결과에 직접 영향을 주는 방식으로 처리하였다. `src/merge_data.py`에서 Steam Store 데이터와 SteamSpy 데이터를 `appid` 기준으로 내부 조인한 뒤, 분석에 반드시 필요한 컬럼이 비어 있는 행을 제거하였다.

처리 방식은 다음과 같다.

| 처리 대상 | 처리 방법 | 이유 |
| --- | --- | --- |
| `appid`, `name`, `is_free`, `price`, `genres`, `owners_avg` 결측 | 행 삭제 | 분석과 병합에 필요한 핵심 컬럼이므로 대체 시 왜곡 가능성이 큼 |
| 리뷰 수가 0인 데이터 | 행 삭제 | `positive_rate` 계산 불가 |
| 무료 게임의 가성비 점수 | `value_score`를 비워 둠 | 가격이 0이므로 가격으로 나누는 가성비 계산이 불가능 |
| 가격 음수 | 확인 후 없음 | 가격 데이터의 기본 유효성 확인 |

정제 과정 이후 최종 데이터에는 `review_count`, `positive_rate`, `value_score` 파생 컬럼을 생성하였다. 다만 무료 게임처럼 가성비 계산이 불가능한 경우에는 `value_score`가 결측으로 남아 있으며, 이는 오류가 아니라 분석 기준에 따른 의도적인 처리이다.

## 4. 데이터 타입 변환

CSV에서 읽어 온 데이터는 문자열로 처리될 수 있으므로 분석 전에 필요한 컬럼을 원하는 타입으로 변환하였다. `src/merge_data.py`와 `src/analysis.py`에서 숫자형 컬럼은 `pd.to_numeric(..., errors="coerce")`를 사용해 숫자형으로 변환하였다.

변환한 주요 컬럼은 다음과 같다.

| 컬럼 | 변환 타입 | 활용 목적 |
| --- | --- | --- |
| `price` | 숫자형 | 가격 비교, 유료 게임 필터링, 가격과 긍정률 상관분석 |
| `owners_avg` | 숫자형 | 평균 소유자 수 비교 |
| `positive` | 숫자형 | 긍정률 계산 |
| `negative` | 숫자형 | 긍정률 계산 |
| `average_forever` | 숫자형 | 평균 플레이타임 분석 |
| `review_count` | 숫자형 | 총 리뷰 수 계산 |
| `positive_rate` | 숫자형 | 사용자 평가 지표 |
| `value_score` | 숫자형 | 장르별 가성비 분석 |
| `is_free` | 불리언 | 무료/유료 게임 그룹 비교 |

특히 `is_free`는 문자열 `"true"`, `"1"`, `"yes"` 등을 불리언 값으로 변환하여 무료 게임과 유료 게임을 정확히 구분할 수 있도록 하였다.

## 5. 필요한 데이터 관계 분석 및 쿼리 작성

분석 목적에 따라 세 가지 관계를 중심으로 데이터를 집계하였다.

### 5-1. 무료 게임과 유료 게임의 인기도 비교

무료 여부(`is_free`)에 따라 데이터를 그룹화한 뒤 평균 소유자 수와 평균 긍정률을 비교하였다.

```python
summary = (
    df.assign(type=df["is_free"].map({True: "Free", False: "Paid"}))
    .groupby("type", as_index=False)
    .agg(
        mean_owners=("owners_avg", "mean"),
        mean_positive_rate=("positive_rate", "mean")
    )
)
```

분석 결과 무료 게임은 149개, 유료 게임은 726개였다. 무료 게임의 평균 소유자 수는 약 10,137,580명으로 유료 게임의 약 4,949,725명보다 높았다. 반면 평균 긍정률은 무료 게임이 약 78.57%, 유료 게임이 약 87.20%로 유료 게임이 더 높았다.

### 5-2. 가격과 사용자 평가의 관계 분석

가격과 긍정률의 관계는 유료 게임만 대상으로 분석하였다. 무료 게임은 가격이 0이므로 가격과 평가의 관계를 분석하는 데 적합하지 않기 때문이다.

```python
paid = df[(df["price"] > 0) & df["positive_rate"].notna()].copy()
price_corr = df.loc[df["price"] > 0, ["price", "positive_rate"]].corr().iloc[0, 1]
```

분석 결과 가격과 긍정률의 상관계수는 약 -0.1385로 나타났다. 이는 가격이 높다고 해서 사용자 평가가 높아지는 경향은 강하지 않으며, 오히려 아주 약한 음의 관계가 있음을 의미한다.

### 5-3. 장르별 가성비 분석

하나의 게임이 여러 장르를 가질 수 있으므로 `genres` 값을 세미콜론 기준으로 분리하여 장르별 행으로 펼쳤다. 이후 장르별 평균 가격, 평균 긍정률, 평균 플레이타임, 가성비 점수를 집계하였다.

```python
summary = (
    paid_genres.groupby("genre", as_index=False)
    .agg(
        game_count=("genre", "size"),
        mean_price=("price", "mean"),
        mean_positive_rate=("positive_rate", "mean"),
        mean_playtime=("average_forever", "mean"),
        mean_value_score=("value_score", "mean"),
        median_value_score=("value_score", "median"),
    )
    .query("game_count >= 5")
    .sort_values("median_value_score", ascending=False)
    .head(12)
)
```

장르별 가성비 순위는 평균값보다 중앙값(`median_value_score`)을 기준으로 정렬하였다. 평균은 일부 극단값의 영향을 크게 받을 수 있으므로, 장르별 대표적인 경향을 비교하는 데 중앙값이 더 적절하다고 판단하였다.

## 6. 최종 결과 검증 및 분석 목적과의 일치 여부

최종 결과는 `data/merged_games.csv`와 `outputs` 폴더의 시각화 파일로 검증하였다. 분석 스크립트 실행 결과 가격과 긍정률의 상관계수는 `-0.1385`로 출력되었고, 다음 시각화 파일이 정상 생성되었다.

| 파일 | 내용 |
| --- | --- |
| `outputs/chart_free_vs_paid.png` | 무료/유료 게임의 평균 소유자 수와 평균 긍정률 비교 |
| `outputs/chart_price_vs_rating.png` | 유료 게임의 가격과 긍정률 산점도 및 회귀선 |
| `outputs/chart_genre_value.png` | 장르별 가성비 상위 장르 |
| `outputs/chart_genre_heatmap.png` | 장르별 가격, 긍정률, 플레이타임, 가성비 지표 비교 |

분석 목적은 Steam 게임의 가격, 인기도, 사용자 평가, 장르 사이의 관계를 확인하는 것이었다. 최종 결과는 이 목적과 일치한다. 무료 게임은 평균 소유자 수가 더 높아 접근성과 인기도 측면에서 강점이 있었지만, 평균 긍정률은 유료 게임이 더 높았다. 또한 가격과 긍정률 사이의 상관관계는 약한 음의 관계로 나타나, 비싼 게임이 반드시 더 좋은 평가를 받는다고 보기 어렵다.

따라서 본 분석을 통해 "무료 게임은 더 많이 보유되는 경향이 있지만 사용자 만족도까지 더 높다고 단정할 수는 없다", "가격은 사용자 평가를 설명하는 핵심 요인이라고 보기 어렵다", "장르별 가성비는 가격, 긍정률, 플레이타임을 함께 고려해야 한다"는 결론을 얻을 수 있었다.
