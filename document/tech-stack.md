## 활용 기술 및 적용 내용

| 요구사항                  | 적용 방법                                                                                      |
| --------------------- | ------------------------------------------------------------------------------------------ |
| pandas DataFrame 활용   | Steam Store 데이터셋, SteamSpy 데이터셋 및 통합 데이터셋을 DataFrame 형태로 관리                                |
| DataFrame 속성 및 메서드 활용 | info(), describe(), isnull(), dropna(), groupby(), sort_values(), corr() 등을 활용하여 데이터 분석 수행 |
| 결측값 처리                | 가격, 장르, 리뷰 수, 플레이타임 등의 결측값을 확인하고 제거                                                        |
| 이상치 처리                | 리뷰 수, 소유자 수, 플레이타임의 극단값을 탐지 및 분석                                                           |
| IQR(사분위 범위) 분석        | 긍정률, 리뷰 수, 평균 플레이타임에 대한 IQR 계산 후 이상치 판별                                                    |
| CSV 파일 저장             | API 수집 결과와 통합 분석 데이터를 CSV 파일로 저장                                                       |
| matplotlib 그래프 1      | 가격과 긍정률의 관계를 나타내는 산점도(Scatter Plot) 생성                                                     |
| matplotlib 그래프 2      | 무료 게임과 유료 게임의 평균 긍정률을 비교하는 막대그래프(Bar Chart) 생성                                             |
| seaborn 시각화           | 장르별 긍정률 분포를 확인하기 위한 박스플롯(Box Plot) 생성                                                      |
